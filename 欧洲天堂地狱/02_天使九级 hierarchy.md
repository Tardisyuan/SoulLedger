# 天使九级 hierarchy — 基督教天使学

基督教天使体系源自《圣经》，经由伪狄奥尼修斯（Pseudo-Dionysius）的《天使等级体系》（Celestial Hierarchy）系统化，形成九级天使的三层结构。

---

## 一、九级天使总览

九级天使分为三个阶层（Triads），每阶三层：

```
                    上帝宝座
                        │
        ┌───────────────┼───────────────┐
        ▼               ▼               ▼
    第一阶层          第二阶层         第三阶层
   （近侍神）        （中层管理）      （执行层）
        │               │               │
   ┌────┼────┐    ┌────┼────┐    ┌────┼────┐
   ▼    ▼    ▼    ▼    ▼    ▼    ▼    ▼    ▼
 炽天使 智天使 座天使  主天使  能天使  权天使  力天使  大天使 天使
Seraphim Cherubim Thrones Dominions Virtues Powers Principalities Archangels Angels
```

---

## 二、第一阶层 — 近侍神（环绕上帝宝座）

### 1. 炽天使（Seraphim）

| 项目 | 内容 |
|------|------|
| 希伯来语 | Seraphim（燃烧者/火热者） |
| 象征 | 上帝之爱 |
| 特征 | 六翼两两相对：一对遮面，一对遮脚，一对飞翔 |
| 职责 | 昼夜不息地赞美上帝，"圣哉！圣哉！圣哉！" |
| 圣经出处 | 以赛亚书 6:1-8 |
| 位置 | 最近上帝宝座 |

> 以赛亚书 6:2："其上有撒拉弗（Seraphim）侍立，各有六个翅膀：用两个翅膀遮脸，两个翅膀遮脚，两个翅膀飞翔。"

### 2. 智天使（Cherubim）

| 项目 | 内容 |
|------|------|
| 希伯来语 | Kerubim（守望者/基路伯） |
| 象征 | 上帝智慧/知识 |
| 特征 | 四翼（以赛亚书6：鹰翅）、满眼发光（以西结书1：四脸） |
| 职责 | 守护上帝宝座，执行神圣命令 |
| 圣经出处 | 创世记 3:24（伊甸园守护者），以西结书 10 |
| 位置 | 上帝宝座两侧 |

> 注：流行文化中常将智天使简化为"小天使"形象，《圣经》原文描述的是极为威严可畏的存在。中世纪艺术中将其描绘为"狮身人面鹰翅"是艺术演绎，非圣经原文。

### 3. 座天使（Thrones / Ophanim）

| 项目 | 内容 |
|------|------|
| 希伯来语 | Ophanim（轮子）/Thrones |
| 象征 | 上帝公义/审判 |
| 特征 | 如轮中套轮，满眼是眼，基路伯在其旁（以西结书1） |
| 职责 | 承载上帝宝座，执行神圣审判 |
| 圣经出处 | 但以理书 7:9，以西结书 1，歌罗西书 1:16 |
| 位置 | 上帝宝座之下 |

> 注：Ophanim（以芬尼姆）和 Thrones 常被等同，但有些神学家认为 Ophanim 是比 Thrones 更高或不同的存在。

---

## 三、第二阶层 — 中层管理者

### 4. 主天使（Dominions / Dominations）

| 项目 | 内容 |
|------|------|
| 拉丁语 | Dominationes |
| 象征 | 上帝主权 |
| 职责 | 管辖下级天使，掌管宇宙秩序 |
| 圣经出处 | 彼得前书 3:22，歌罗西书 1:16 |

### 5. 能天使（Virtues）

| 项目 | 内容 |
|------|------|
| 拉丁语 | Virtutes |
| 象征 | 上帝德能/奇迹 |
| 职责 | 掌管自然界奇迹，执行神迹 |
| 圣经出处 | 彼得前书 3:22 |

### 6. 权天使（Powers）

| 项目 | 内容 |
|------|------|
| 拉丁语 | Potestates / Authorities |
| 象征 | 上帝权能 |
| 职责 | 对抗邪恶势力，保护信徒 |
| 圣经出处 | 歌罗西书 1:16，以弗所书 3:10 |

> 注：权天使被认为是与恶魔、邪灵作战的天使军团。

---

## 四、第三阶层 — 执行层

### 7. 掌权天使（Principalities）

| 项目 | 内容 |
|------|------|
| 拉丁语 | Principatus / Principalities |
| 象征 | 民族/国家守护 |
| 职责 | 守护民族、国家、城市，引导人类历史 |
| 圣经出处 | 但以理书 10:13（米迦勒为一国大君） |

> 注：Principalities 常被误译为"大天使"，但其职能与 Archangels（总领天使）不同。中文"掌权天使"或"主治天使"更为准确。 |

### 8. 总领天使（Archangels）

| 项目 | 内容 |
|------|------|
| 希伯来语 | Sarafim（领袖） |
| 象征 | 上帝首席使者 |
| 职责 | 传递重大启示，执行重要使命 |
| 圣经出处 | 帖撒罗尼迦前书 4:16（主由天使声中降临） |

> 注：Archangels 与 Principalities（掌权天使）是不同层级。Archangels 是上帝的首席使者（如米迦勒、加百列），Principalities 是守护民族国家的执政天使。 |

**被提名的大天使**：
| 大天使 | 职能 |
|--------|------|
| 米迦勒（Michael） | 战神，击败撒旦，保护以色列 |
| 加百列（Gabriel） | 报信天使，向马利亚报喜 |
| 拉斐尔（Raphael） | 治愈天使（次经《托比特》） |
| 乌列尔（Uriel） | 智慧天使（次经《以斯拉续书》） |

### 9. 天使（Angels）

| 项目 | 内容 |
|------|------|
| 希伯来语 | Mal'akh（使者） |
| 象征 | 上帝普通使者 |
| 职责 | 引导个人灵魂，向个人传递信息 |
| 圣经出处 | 希伯来书 13:2（款待旅人） |

## 五、天使与天堂层级映射

天堂的荣耀等级与天使层级相对应：

| 阶层 | 天使 | 接近上帝程度 | 荣耀等级 |
|------|------|-------------|---------|
| 第一阶 | Seraphim/Cherubim/Thrones | 最近 | 最高 |
| 第二阶 | Dominions/Virtues/Powers | 中间 | 中等 |
| 第三阶 | Principalities/Archangels/Angels | 最远 | 基础 |

## 六、堕落天使 — 撒旦与恶魔

|| 堕落者 | 原身份 | 堕落原因 |
|--------|--------|---------|
| 撒旦/路西法（Lucifer） | 明光之星（赛14:12），天使等级圣经未明确记载 | 骄傲，欲与上帝同等 |
| 堕落天使1/3 | 各等级天使 | 跟随撒旦反叛 |

> 注：撒旦原为炽天使（Seraphim）是流行说法，《圣经》仅称其为"明亮之星"（赛14:12）或"堕落的守护天使"（结28:13-17），未明确其原始天使等级。

**地狱层级结构**（部分传统说法）：
- 撒旦（Satan）→ 路西法，在地狱最深处
- 七大魔王：分别代表七宗罪

## 七、天使层级与管理系统映射

| 天使层级 | 系统概念 |
|---------|---------|
| 炽/智/座天使 | 董事会/最高决策层 |
| 主/能/权天使 | 部门总监 |
| 掌权/总领/天使 | 分支机构负责人 |
| 天使 | 一线执行人员 |
| 米迦勒 | CEO/COO（执行总裁） |
| 加百列 | CMO/COO（传达指令） |
