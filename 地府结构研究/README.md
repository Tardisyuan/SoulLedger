                                                  │
                            ┌─────────────────────┴─────────────────────┐
                            │                                           │
                        无罪 → 直接投胎                            有罪 → 量刑
                                                              │
                                                    ┌───────────┴───────────┐
                                                    │                       │
                                            轻罪 → 地狱服刑                  重罪 → 阿鼻地狱
                                                    │                       │
                                                    └──→ 刑满 → 投胎 ←──────┘
                                                              │

## 冥律核心原则

1. **因果报应**：善有善报，恶有恶报
2. **功过相抵**：善行可减轻恶行刑罚
3. **寿数既定**：寿命由生死簿注定
4. **审判公正**：有起诉→辩护→判决→上诉的完整程序
5. **刑罚适当**：刑罚与罪行相匹配

> 法律依据：《冥律》（又称《阴律》《玉历宝钞》）
