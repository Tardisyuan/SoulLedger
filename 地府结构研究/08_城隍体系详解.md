# 城隍体系详解

城隍是中国传统文化中城市的守护神，相当于阴间的"地方官"，掌管一方阴阳两界的事务。

---

## 一、城隍信仰起源

- **起源**：周朝天子祭祀八蜡中的水庸神
- **发展**：汉代开始有城隍祠，唐代普及，宋代以后遍布全国
- **制度化**：明洪武年间朱元璋下旨，城隍祭祀列入国家祀典

---

## 二、城隍层级体系

城隍按行政级别分为四级，与人间地方行政体系对应：

| 级别 | 名称 | 爵位 | 管辖范围 |
|------|------|------|----------|
| 省级 | 都省城隍 | 公爵（威灵公） | 管辖省 |
| 府级 | 府城隍 | 侯爵（绥靖侯） | 管辖府 |
| 州级 | 州城隍 | 伯爵（显佑伯） | 管辖州 |
| 县级 | 县城隍 | 子爵（显正伯） | 管辖县 |

> 城隍爷级别越高，下属二十四司配置越齐全。

---

## 三、二十四司（城隍的下属机构）

城隍神务繁重，仿古时六部（吏、户、礼、兵、刑、工），每部各设四司，共二十四司：

### 3.1 阴阳司（总务）

| 司名 | 职能 |
|------|------|
| 阴阳司 | 辅佐城隍爷督察阴阳两界的总管 | 掌理阳世之善恶，记录功过 |
| 说明 | 二十四司之首，相当于秘书长 |

### 3.2 吏部（人事）

| 司名 | 职能 |
|------|------|
| 功曹司 | 考核记录亡魂生前功过 |
| 典籍司 | 掌管亡魂名册簿典 |
| 记簿司 | 登录亡魂生前言行 |
| 都官司 | 调度鬼卒执行任务 |

### 3.3 户部（民政）

| 司名 | 职能 |
|------|------|
| 财帛司 | 管理阳间纸钱、元宝接收 |
| 土地司 | 管辖境内土地事务 |
| 粮秣司 | 管理粮食储存分配 |
| 考劫司 | 记录人在阳间寿夭劫数 |

### 3.4 礼部（礼仪）

| 司名 | 职能 |
|------|------|
| 祭祀司 | 掌管祠庙祭祀事务 |
| 仪注司 | 制定祭祀礼仪规范 |
| 祸福司 | 宣告祸福吉凶 |
| 冥衣司 | 管理亡魂衣物冥器 |

### 3.5 兵部（军事）

| 司名 | 职能 |
|------|------|
| 兵杖司 | 掌管刑罚器具 |
| 刑讯司 | 审讯罪犯亡魂 |
| 拘捕司 | 缉拿游魂散鬼 |
| 巡察司 | 巡查阳间善恶 |

### 3.6 刑部（司法）

| 司名 | 职能 |
|------|------|
|| 刑罚司 | 执行各种刑罚 |
|| 案簿司 | 整理案件卷宗 |
|| **权典司** | 裁决疑难案件 |
| 续命司 | 管理寿夭续命事务 |

### 3.7 工部（工程）

| 司名 | 职能 |
|------|------|
|| 营造司 | 建造修缮阴间建筑 |
|| 桥道司 | 管理奈何桥等通道 |
|| **祠祭司** | 掌管祠庙祭祀、亡魂沐浴洁身之事 |
| 输运司 | 押送亡魂来往各站 |

## 四、城隍的主要职能

### 4.1 阴间地方官职能

| 职能 | 说明 |
|------|------|
| 护城安民 | 保护城市安全，防御灾祸 |
| 管辖亡魂 | 接收、管理本地区亡魂 |
| 监察善恶 | 记录境内居民善恶行为 |
| 缉拿鬼魂 | 派遣黑白无常等勾魂 |
| 上报下达 | 向地府上报，传达审判结果 |

### 4.2 文武判官（城隍的左右手）

| 判官 | 职能 |
|------|------|
| 文判官 | 掌管文书、记录、审核 |
| 武判官 | 协助缉拿、押送、刑罚执行 |

## 五、城隍与上级的关系

```
后土皇地祇
    │
    ├──→ 酆都大帝 ──→ 五方鬼帝 ──→ 十殿阎罗 ──→ 四大判官
    │
    └──→ 东岳大帝
              │
              └──→ 城隍（地方官）
                        │
                        ├──→ 土地公（乡镇级）
                        │
                        └──→ 二十四司
```

**管辖关系**：
- 城隍受东岳大帝和酆都大帝双重管辖
- 城隍可直接向十殿阎罗报送案件
- 城隍的审判结果可上诉至十殿阎罗

---

## 六、城隍的诞辰与祭祀

| 项目 | 内容 |
|------|------|
| 诞辰 | 各城隍不同，如福州城隍五月初五 |
| 祭祀日 | 明洪武后规定每年清明、七月十五、十月初一祭祀 |
| 庙会 | 城隍出巡（抬城隍像游街） |

---

## 七、城隍与土地公的区别

| 对比项 | 城隍 | 土地公 |
|--------|------|--------|
| 级别 | 较高（县级/市级/省级） | 最低（乡镇级） |
| 管辖范围 | 整个城市 | 一方土地 |
| 职能 | 阴间地方官，审判职能 | 管理一方土地和亡灵 |
| 祭祀规格 | 较高，国家祀典 | 较低，民间小祭 |
