# 地府管理系统 — 整体架构文档

## 一、地府定位

地府是中国传统神话体系中管理亡魂、审判善恶、执行因果报应的幽冥世界。融合了道教、佛教及民间信仰，是一个多层次、功能完整的"阴间政府"。

---

## 二、地府层级架构（从上到下）

### 🔺 第一层：最高神祇

| 神祇 | 职责 | 备注 |
|------|------|------|
| **后土皇地祇** | 阴阳两界总源头，掌生死轮回、大地与幽冥 | 被尊为"大地之母"，地府创始人 |
| **酆都大帝**（北阴大帝） | 统领整个阴间，位于罗酆山（酆都城） | 道教阴间最高神，类似"阴间玉皇大帝" |
| **东岳大帝**（泰山神） | 掌管万物出生大权，护佑轮回转世众生 | 泰山的山神，冥界保护者 |
|| **地藏王菩萨** | 度化六界一切众生，包括恶鬼及因果报应巨大亡魂 | 发愿"地狱不空，誓不成佛"，坐骑为谛听 |
| **谛听** | 地藏王菩萨坐骑（从属） | 辨别亡魂善恶真伪，协助超度 | 祥瑞之兽，能洞悉一切善恶真伪 |

> **谛听说明**：谛听为地藏王菩萨坐骑，非独立神祇。其主要来源于中国民间传说（《西游记》），非佛教原典记载。在系统中可作为辅助角色处理。

> **层级关系**：后土皇地祇 → 酆都大帝/东岳大帝 → 十殿阎王
>
> **说明**：酆都大帝主管阴间行政与审判；东岳大帝主管万物出生；地藏王菩萨主管超度与救度。三者各司其职，并无绝对上下级关系（部分文献记载地藏王菩萨"统率十殿阎王"）。

> **法律依据**：地府审判依据《冥律》（又称《阴律》《玉历宝钞》），详见《11_地府审判制度与冥律.md》

### 🔶 第二层：五大鬼域（五方鬼帝）

镇守东、西、南、北、中五大鬼域，统率鬼兵，管辖不同方向的亡魂。

|| 鬼帝 | 镇守位置 | 主要职责 |
||------|----------|----------|
|| **神荼**（东方鬼帝） | 桃止山（鬼门关） | 镇守鬼门关，驱逐恶鬼 |
|| **郁垒**（东方鬼帝） | 鬼门关 | 与神荼共同把守鬼门关 |
|| **赵文和/王真人**（西方鬼帝） | 嶓冢山 | 管辖西方鬼域 |
|| **杜子仁**（南方鬼帝） | 罗浮山 | 管辖南方鬼域，镇压凶魂 |
|| **张衡/杨云**（北方鬼帝） | 罗酆山 | 坐镇酆都，掌管地狱刑罚 |
|| **中央鬼帝**（周文） | 抱犊山 | 协调五方鬼帝，维持阴阳平衡 |

> **注**：东方有神荼、郁垒两位鬼帝（特有配置），其余四方各一位或存在候选人争议，此为历史文献记载的差异，不影响系统运作。中央鬼帝文献记载较少，名称有争议。

### 🔷 第三层：十殿阎罗（核心审判机构）

地府的**核心审判机构**，类似人间的十个中级人民法院。十殿阎罗分掌十殿地狱，按职能分工。

| 殿位 | 阎王（姓名） | 职责 | 管辖地狱 |
|------|-------------|------|----------|
| **第一殿** | 秦广王（蒋子文） | 初判亡魂善恶，善者直接转世，恶者移交下一殿 | 无（分流殿） |
| **第二殿** | 楚江王（厉温） | 审判伤人肢体、奸盗杀生者 | 剥衣亭、刀林地狱 |
| **第三殿** | 宋帝王（余懃） | 审判不孝、忤逆尊长者 | 黑绳大地狱、剥皮地狱 |
| **第四殿** | 五官王（吕岱） | 审判欺诈、逃税、贪赃者 | 秤钩地狱、血池地狱 |
| **第五殿** | 阎罗王（包拯） | 复审前四殿案件，著名"森罗殿"之主（原型为包拯，称"阎罗包老"） | 叫唤大地狱、诛心地狱 |
| **第六殿** | 卞城王（毕元宾） | 审判怨天尤人、亵渎神明者 | 沸汤地狱 |
| **第七殿** | 泰山王（董和） | 审判堕胎、纵欲、离间骨肉者 | 热恼大地狱 |
| **第八殿** | 都市王（黄中庸） | 审判不敬先人、浪费粮食者 | 热恼闷锅地狱 |
| **第九殿** | 平等王（陆游） | 审判杀人放火、谋财害命者 | 阿鼻地狱（无间地狱） |
| **第十殿** | 转轮王（薛礼） | 最终核定投胎去向（六道轮回），发放"转世批文" | 根据善恶决定人畜鬼等转世身份 |

### 🔹 第四层：行政与司法官员

#### 四大判官

负责协助阎王审理亡魂、执行刑罚，并掌管生死簿。象征司法公正。

| 判官 | 职责 | 原型/来源 |
|------|------|-----------|
| **崔珏**（崔府君） | 掌管生死簿，主判善恶，决定寿夭 | 历史人物 |
| **钟馗** | 专管厉鬼、凶煞，降妖除魔，兼管"罚恶司" | 传说人物 |
| **魏征** | 负责"察查司"，稽查冤假错案，纠正阎王误判 | 历史人物（唐朝丞相） |
| **陆之道**（陆判） | 主管"阴律司"，修订阴间法律，监督鬼差执法 | 传说人物 |

> **级别参考**：四大判官与六案功曹同级，位于十殿阎王殿之下。

#### 六案功曹

地府的文职官员，类似于天庭或阴间的"秘书"或"书记官"。

| 功曹类型 | 职责 |
|----------|------|
| **天曹** | 记录人在天上的善恶行为（如祭祀、修道、行善积德） |
| **地曹** | 记录人在阳间的行为（如孝道、忠义、作恶） |
| **神曹** | 负责与诸神沟通，传递文书（如上报天庭、下达地府） |
| **鬼曹** | 管理鬼卒、执行阴司命令（如勾魂、押送亡魂） |

> **判官抓人，功曹记录**，最后交由阎王殿审判。

---

### 🔸 第五层：执行层（鬼差鬼卒）

负责缉拿鬼魂、押送亡魂、协助赏善罚恶。

#### 黑白无常
- **职责**：听从城隍指示，来阳间勾魂
- **级别**：七爷（白无常）、八爷（黑无常）
- **特点**：负责难度相对较低的勾魂任务

#### 牛头马面
- **职责**：专门押解人犯前去审判，协助阎王或城隍
- **特点**：地位高于黑白无常，处理更凶恶的亡魂

#### 其他鬼差
- **枷锁将军**：负责给亡魂戴上枷锁
- **日游神/夜游神**：在人间巡查
- **孟婆**：在奈何桥边熬孟婆汤，消除记忆

---

### ⬜ 辅助层：地方神祇

|| 神祇 | 职责 | 层级 |
||------|------|------|
|| **城隍** | 城市的守护神，管理本城亡魂，相当于阴间的"地方官" | 县级/市级 |
|| **土地公** | 最低级别的阴间神祇，管理一方土地 | 乡镇级 |

> **注**：辅助层中钟馗已列入第四层判官，此处不再重复列出。

## 三、地府核心地点

| 地点 | 功能 |
|------|------|
| **酆都城**（罗酆山） | 阴间的行政中心，酆都大帝统领 |
| **鬼门关** | 阳间进入阴间的入口，由神荼、郁垒把守 |
| **奈何桥** | 亡魂投胎的必经之路，有孟婆熬汤消除记忆 |
| **望乡台** | 亡魂最后眺望人间的地方 |
| **生死簿存储地** | 记录所有生灵的生死年限 |
| **森罗殿** | 第五殿阎罗王的审判大厅 |

## 四、附：地狱体系

### 十殿地狱（对应十殿阎王）

每殿都有对应的专属地狱，刑罚根据亡魂生前罪行分配。

### 十八层地狱（佛教体系）

| 层数 | 地狱名称 | 惩罚对象 |
|------|----------|----------|
| 第一层 | 拔舌地狱 | 挑拨离间、诽谤害人、油嘴滑舌、说谎骗人 |
| 第二层 | 剪刀地狱 | 唆使寡妇改嫁者 |
| 第三层 | 铁树地狱 | 离间骨肉、挑拨父子兄弟不和者 |
| 第四层 | 孽镜地狱 | 犯罪后不立即承认、企图抵赖者 |
| 第五层 | 蒸笼地狱 | 说长道短、传播是非、挑拨离间者 |
| 第六层 | 铜柱地狱 | 故意纵火或毁灭罪证者 |
| 第七层 | 刀山地狱 | 杀生者（尤其是故意杀害） |
| 第八层 | 冰山地狱 | 不孝、堕胎、遗弃子女者 |
| 第九层 | 油锅地狱 | 卖淫嫖娼、盗贼抢劫、欺善凌弱者 |
| 第十层 | 牛坑地狱 | 伤害动物、虐待生灵者 |
| 第十一层 | 石压地狱 | 溺婴、弃婴、杀害婴儿者 |
| 第十二层 | 舂臼地狱 | 浪费五谷、糟蹋粮食者 |
| 第十三层 | 血池地狱 | 贪官污吏、欺压百姓者 |
| 第十四层 | 枉死地狱 | 自杀、轻生者 |
| 第十五层 | 磔刑地狱 | 谋财害命者 |
| 第十六层 | 火山地狱 | 损公肥私、贪污受贿者 |
| 第十七层 | 石磨地狱 | 诱骗强迫他人卖淫者 |
| 第十八层 | 刀锯地狱 | 偷工减料、欺骗他人者 |

> 每一层地狱比前一层增苦二十倍，增寿一倍。

## 五、管理系统建模要点

如果要用地府结构做管理系统，可以参考以下映射：

| 地府元素 | 对应现实系统概念 |
|----------|------------------|
| 酆都大帝 | 董事长/CEO |
| 十殿阎罗 | 十个业务部门/分公司 |
| 四大判官 | 核心高管（法务、审计、风控） |
| 六案功曹 | 行政文秘部门 |
| 黑白无常/牛头马面 | 执行层/外勤部门 |
| 城隍/土地公 | 地方分支机构 |
| 十八层地狱 | 惩罚机制/风控等级 |
| 生死簿 | 核心数据库 |
| 投胎转世 | 业务流程终点 |
| 枉死城 | 特殊关押设施（非刑狱，用于收容非命死亡的亡魂） |
| 忘川河/奈何桥 | 状态流转节点（记忆消除、投胎分叉口） |
