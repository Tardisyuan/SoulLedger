# 杜阿特冥界地理详解 — 古埃及地下世界

杜阿特（Duat）是古埃及神话中的地下世界，是亡灵从死亡到抵达真理大厅接受审判的旅程区域。

---

## 一、重要说明

> **本文档性质**：传统《亡灵书》《金字塔铭文》《石棺铭文》中对杜阿特地理的描述是**连续的地下区域**，并非严格的"十二门"划分。"十二门"结构是**现代学者根据各章节描述重构**的，并非古典文献的统一记载。

**以下内容的不确定性**：
- 各门名称：大多由现代学者根据描述推测命名，非原典明确记载
- 门数量：《亡灵书》各版本说法不一（有的说7个门廊，有的说更多）
- 守卫名称：大多数是推测或简化描述

**正确的理解方式**：将本文档视为"基于文献描述的现代重构"，而非"古典记载的准确复现"。

### 2.1 三层结构

杜阿特分为上中下三层：
- **上层**：太阳落山进入之处
- **中层**：亡灵主要旅程区

## 四、太阳船之旅

亡灵旅程的一部分是乘坐太阳船（Solar Barque）与太阳神拉（Ra）同行：

| 夜间时段 | 太阳船位置 | 经历区域 |
|---------|-----------|---------|
| 黄昏 | 进入杜阿特 | 入口区域 |
| 深夜 | 穿越中下层 | 火焰湖、黑暗区 |
| 黎明前 | 接近真理大厅 | 下层区域 |
| 黎明 | 离开杜阿特 | 重生 |

> 太阳每日在杜阿特中穿行一夜，第二天早晨重新升起——这是古埃及神话中太阳每日重生的象征。

## 五、阿米特的位置（重要澄清）

> **阿米特（Ammit）不在杜阿特的任何一门或区域之中**

**正确位置**：阿米特在**真理大厅（Hall of Ma'at）**的审判现场等待。

**审判流程**：
1. 亡灵抵达真理大厅
2. 心脏与玛特羽毛称重
3. 如果天平平衡 → 亡灵通过审判，前往芦苇原
4. 如果心脏过重 → 阿米特立即上前吞噬，灵魂彻底消亡

阿米特是审判的**结果执行者**，不是冥界旅程的**关卡守卫**。

## 六、通关要素

亡灵能成功通过杜阿特抵达审判，需要：

1. **身体完整**：木乃伊化成功，身体各部分齐全
2. **咒语保护**：携带或记忆《亡灵书》中的保护咒语
3. **正确自我介绍**：向各区域守卫正确报出自己与神明的关联
4. **心脏轻盈**：审判时心脏重量不超过玛特羽毛
5. **知道神名**：掌握各区域神明的名字
