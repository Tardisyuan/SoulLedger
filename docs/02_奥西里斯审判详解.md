# 奥西里斯审判 — 心脏称重详解

奥西里斯审判（Osirian Judgment）是古埃及最核心的死后审判机制，发生在真理大厅（Hall of Ma'at），是亡灵能否进入芦苇原的关键。

---

## 一、审判准备

### 1.1 死亡到审判的旅程

```
死亡（心脏停止跳动）
    │
    ▼
制作木乃伊（保存尸体）
    │
    ▼
制作木乃伊（75天后完成）
    │
    ▼
亡灵（Ba和Ka）出发旅行
    │
    ▼
通过杜阿特十二门
    │
    ▼
抵达真理大厅
```

### 1.2 灵魂构成（古埃及信仰）

| 部分 | 含义 | 说明 |
|------|------|------|
| Ba（巴） | 灵魂 | 人形鸟，可在天空飞翔 |
| Ka（卡） | 生命力量/分身 | 需要食物供奉 |
| Akh（阿赫） | 升华灵魂 | 死后与神明合一 |
| Ib（伊布） | 心脏 | 记忆与意志所在，审判的关键 |
| Sheut（舍特） | 影子 | 永远跟随 |

## 三、心脏称重仪式

### 3.1 称重过程

1. **摆放天平**：托特调整天平
2. **放置心脏**：将亡灵的 Ib（心脏）放在天平左侧
3. **放置羽毛**：将玛特（Ma'at）的羽毛放在天平右侧
4. **观察结果**：
   - 平衡 = 通过审判
   - 倾斜（心脏过重）= 罪恶深重
   - 摇摆不定 = 需进一步审查

### 3.2 审判关键：心胜于羽毛

古埃及人认为：
- 心脏记录了一生的所有行为（善恶）
- 心脏是智慧和记忆的所在
- 审判时，心脏必须与玛特的真理羽毛同等轻盈

**影响审判的行为**：
| 加重心脏（罪恶） | 减轻心脏（良善） |
|-----------------|-----------------|
| 谋杀 | 善待动物 |
| 偷窃 | 施舍穷人 |
| 通奸 | 照顾孤儿 |
| 欺诈 | 诚实守信 |
| 贪婪 | 敬拜神明 |
| 傲慢 | 谦逊待人 |
| 浪费谷物 | 珍惜食物 |

## 五、审判结果

### 5.1 通过审判

**条件**：心脏重量 ≤ 羽毛重量

**结果**：
1. 托特宣布：通过审判
2. 伊西斯和奈芙蒂斯庆祝
3. 亡灵获得进入芦苇原（Aaru）的资格
4. 与奥西里斯一起永生

### 5.2 审判失败

**条件**：心脏重量 > 羽毛重量

**结果**：
1. 阿米特立刻上前
2. 吞噬不合格的灵魂
3. 灵魂彻底消亡（第二死）
4. 无法进入来世

### 5.3 摇摆不定

**条件**：天平摇摆不定

**结果**：
1. 托特记录详细审查
2. 可能需要更多证据
3. 延迟审判
