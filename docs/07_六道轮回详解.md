# 六道轮回详解

六道轮回是佛教因果报应体系的核心概念，众生依生前所造业力，在六种生命形态中循环转世。

---

## 一、六道总览

| 类别 | 道别 | 特质 | 苦乐程度 |
|------|------|------|----------|
| **三善道** | 天道 | 享乐最多，福报最大 | 最乐 |
| | 人道 | 苦乐参半，最好修行 | 苦乐参半 |
| | 阿修罗道 | 有天福无天德，斗争不断 | 有苦有乐 |
| **三恶道** | 畜生道 | 愚痴受苦，互相吞啖 | 苦 |
| | 饿鬼道 | 常受饥渴恐怖 | 极苦 |
| | 地狱道 | 受各种极刑，生命漫长 | 最苦 |

> 轮回转世依"业力"为因，业力分十善（善业）和十恶（恶业）。

---

## 二、三善道详解

### 天道（最上一层）

**特征**：
- 身体高大（最矮九十丈，最高二十五万六千里）
- 寿命极长（最短人间九百岁，最长八万大劫）
- 欲乐无穷（有饮食、男女之欲或禅定之乐）
- 具足神通威德

**三界**：

| 界别 | 层数 | 特点 |
|------|------|------|
| 欲界 | 六层天 | 有饮食男女之欲 |
| 色界 | 十八层天 | 离欲，存身形楼阁 |
| 无色界 | 四层天 | 无身形，唯存神识 |

**欲界六天**：
1. 四天王天（持国、增长、广目、多闻四大天王）
2. 忉利天（三十三天，玉帝所居）
3. 夜摩天
4. 兜率天
5. 化乐天
6. 他化自在天（魔王波旬所居）

**生天条件**：上品十善业 + 禅定功夫

**命终危机**：五衰相现（头上花萎、腋下出汁、衣裳垢染、身体臭秽、不乐本座）

---

### 人道（四洲分布）

**生人条件**：中品十善业因

**四大部洲**：

| 部洲 | 人寿 | 特点 |
|------|------|------|
| 东胜神洲 | 250岁 | 身体中等 |
| 南瞻部洲 | 100岁 | 即吾人所居地球 |
| 西牛贺洲 | 500岁 | 多牛马羊等 |
| 北俱卢洲 | 1000岁 | 寿命最长但无佛法 |

**人道苦乐**：
- 三苦：苦苦、坏苦、行苦
- 八苦：生、老、病、死、爱别离、怨憎会、求不得、五阴识盛
- 苦乐参半 = 最好修行之地

**佛陀出生在人间，成佛也在人间**，故称"人身难得"。

---

### 阿修罗道

**生因**：下品十善业，但常怀忌慢之心

**特征**：
- 男丑女端（故名"无端正"）
- 有天福无天德
- 性好斗争
- 分布在天、人、畜生、饿鬼四道

**处境**：
- 虽享福报但常被天、人歧视
- 嗔恨心重，战斗不断
- 虽有神通但不得解脱

---

## 三、三恶道详解

### 畜生道（血途）

**生因**：中品五逆十恶（愚痴、贪欲、爱见为根、悭贪为业）

**分布**：
- 胎生（母胎中孕育，如猫狗）
- 卵生（蛋中孵化，如鸡鸭鸟雀）
- 湿生（湿暖气生，如龙等）
- 化生（变化而生）

**苦难**：
- 互相残杀、弱肉强食
- 大鱼吃小鱼，小虫吃大虫
- 为人所蓄养、宰割、驱役
- 寿命长短悬殊（有朝生暮死者，有活千百岁者）

---

### 饿鬼道（刀途）

**生因**：下品五逆十恶（欲贪不息、痴想横生）

**分布**：
- 居海中
- 居山林树下
- 居旷野林中
- 或人形或似兽形

**三类九种**：

| 类别 | 种类 | 特征 |
|------|------|------|
| 无财鬼 | 焰口鬼、针咽鬼、臭口鬼 | 不得饮食 |
| 少财鬼 | 针毛鬼、臭毛鬼、大瘿鬼 | 少得饮食 |
| 多财鬼 | 得弃鬼、得失鬼、势力鬼 | 多得饮食（如城隍、土地公等民间神） |

**苦难**：
- 不闻浆水之名，见水如脓血
- 难有饮食充饥
- 多刀杖逼迫，恐怖非常
- 寿命五百岁（以人间一月为一日计）

---

### 地狱道（火途）

**生因**：上品五逆十恶（杀父、杀母、杀罗汉、出佛身血、破和合僧）或谤法破戒

**分类**：

| 类别 | 说明 |
|------|------|
| 八热地狱 | 烈火猛烧 |
| 八寒地狱 | 寒冰煎迫 |
| 孤独地狱 | 山间、水中、旷野 |
| 近边地狱 | 八热地狱边缘 |
| 无间地狱（阿鼻） | 最苦，永远不得超生 |

**无间地狱五种无间**：
1. 时无间（受苦无停歇）
2. 形无间（身躯遍满其中）
3. 苦无间（苦具同时加身）
4. 果无间（罪因次第受报）
5. 命无间（寿命未尽不得死）

**寿命**：极长，初入等活地狱寿同四天王天五百岁（以四天王天五百岁为一日计），层层递增至无间地狱，寿长一中劫。

---

## 四、投胎转世规律

| 今生行善 ↓ / 造恶 ↑ | 来世去处 |
|---------------------|----------|
| 广修上品十善 | 天道 |
| 修持中品十善 | 人道 |
| 修下品十善（常怀忌慢） | 阿修罗道 |
| 行中品十恶（愚痴贪欲） | 畜生道 |
| 行下品十恶（欲贪不息） | 饿鬼道 |
| 行上品五逆十恶 | 地狱道 |
