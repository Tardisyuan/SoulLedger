# 跨文明灵魂管理系统 — 资料汇总

本项目旨在构建一个融合多文明神话体系的灵魂管理系统，涵盖中国地府、欧洲天堂地狱、埃及冥界等主要文明对死后世界的想象。

## 三大文明体系概览

### 中国地府体系
- **核心概念**：因果报应、六道轮回、十殿审判
- **最高神祇**：后土皇地祇、酆都大帝、东岳大帝、地藏王菩萨
- **审判机制**：十殿阎王分级审判，生死簿记录善恶
- **轮回系统**：天道、人道、阿修罗、畜生、饿鬼、地狱六道
- **代表典籍**：《玉历宝钞》《地藏十王经》

### 欧洲天堂地狱体系
- **核心概念**：原罪、救赎、末日审判
- **主要分支**：
  - 基督教（天主教）：天堂九级天使、炼狱、七宗罪
  - 希腊神话：冥界（Hades）、塔尔塔罗斯（Tartarus）、极乐岛（Elysium）
  - 北欧神话：瓦尔哈拉（Valhalla）、海姆冥界（Hel）、福尔克范格（Fólkvangr）
- **审判机制**：生前信仰与善行综合评定
- **代表典籍**：《圣经》《神曲》《埃达经》

### 埃及冥界体系
- **核心概念**：心脏称重、玛特羽毛审判、来世之旅
- **核心神祇**：奥西里斯（Osiris）、阿努比斯（Anubis）、托特（Thoth）、阿米特（Ammit）
- **冥界地理**：杜阿特（Duat）十二门
- **审判机制**：心脏与玛特羽毛称重
- **代表典籍**：《亡灵书》（Book of the Dead）

## 跨文明灵魂管理系统设计要点

### 通用实体抽象

| 跨文明概念 | 中国 | 欧洲 | 埃及 |
|-----------|------|------|------|
| 灵魂存储 | 生死簿 | 末日账册（Book of Life） | 心脏记录 |
| 记忆消除 | 孟婆汤 | 忘川河（Lethe） | 芦苇原忘却是 |
| 审判核心 | 十殿阎王 | 上帝/末日审判 | 奥西里斯+42神 |
| 终极惩罚 | 十八层地狱/阿鼻地狱 |地狱七层/九圈 | 阿米特吞噬 |
| 最终归宿 | 六道轮回 | 天堂/地狱/炼狱 | 芦苇原/极乐世界 |
| 引渡使者 | 黑白无常 | 赫尔墨斯（Hermes） | 阿努比斯引导 |

### 统一善恶标准设计

三大文明善恶标准不同，系统需建立可配置的"善恶积分通兑"机制：
- 中国：功德积分（善行-恶行）
- 欧洲：原罪状态 + 末日审判积分
- 埃及：心脏重量 vs 玛特羽毛
